# Student Feedback Sentiment Analysis

## Overview
An NLP-based analysis of student survey feedback, applying text preprocessing,
sentiment classification, and insight extraction to unstructured qualitative data.

## Problem
Raw survey responses are inconsistent, noisy, and difficult to interpret at scale.
This project builds a systematic pipeline to clean, classify, and surface
actionable themes from student feedback data.

## Approach
- Data cleaning and preprocessing pipeline
- Exploratory Data Analysis (EDA)
- Sentiment classification using NLP techniques
- Manual output validation before drawing conclusions
- Structured insight extraction from unstructured text

## Tools Used
Python, Pandas, NumPy, NLTK / TextBlob / VADER, Matplotlib, Seaborn, Jupyter Notebook

## Key Findings
- Surfaced 3 distinct sentiment themes from survey responses
- Identified key satisfaction and dissatisfaction patterns across respondents
- Produced structured, plain-language findings from raw qualitative input

## Relevance to NLP Work
This project is directly applicable to mixed-language and qualitative narrative
analysis — including classification of needs, barriers, and themes from
community-level survey and interview data.

## Files
